Manejo de variables, tipos de datos y expresiones regulares en JS

Manejo de variables: El manejo de variables como nombres simbólicos para valores en tu aplicaciones. Los nombres de la variables, llamados identificadores, se ajustan a ciertas reglas. Un identificador de js debe comenzar con una letra, un guion bajo o un signo de dólar.

Tipos de datos, En java script existen 7 tipos de datos:

  1. String Cadenas de texto.
  2. number valores numericos.
  3. boolean representa una entidad logica y puede tener dos valores:true o false.
  4. null es un valor asignado tiene valor de "no valor".
  5. undefined una variable a la que no se le ha asignado ningun valor tiene el valor undefined.
  6. symbol nuevo en ECMAScript 2015.
  7. object un valor en memoria al que podemos acceder por un identificador.
  • Expresiones regulares: hay 2 formas: Usando una expresión regular literal, que consiste en un patrón encerrado entre barras, como sigue: Let re = /ab+c/; • Las expresiones regulares literales proporcionan la compilación de la expresión regular cuando se carga el script. Si la expresión regular permanece constante, su uso puede mejorar el rendimiento. • O llamando a la función constructora del objeto RegExp, de la siguiente manera: Let re = new regexp (ab+c) ;
  • El uso de la función constructora proporciona una compilación en tiempo de ejecución de la expresión regular. Usa la función constructora cuando sepas que el patrón de la expresión regular cambiará, o no conoces el patrón y lo obtienes de otra fuente, como la entrada del usuario.

Operadores y conversión entre tipos

Un operador de conversión no es más que un método operator que recibe un tipo por parámetro y devuelve un objeto de otro tipo. Las palabras reservadas implicit y explicit nos permite indicar si la conversión va a ser implícita o explícita.

Estructuras de control JS

Las estructuras de control de JS son las siguientes: IF/ELSE: if (nota >= 5) { console.log (“Has aprobado”); } else { console.log (“Has desaprobado”); } Multiples ELSE: if (nota < 5) { console.log (“Has desaprobado”); } else if (nota==5 ) { console.log (“Has sacado un: suficiente”); } else if (nota==6 ) { console.log (“Has sacado un: bien”); } SWITCH: switch (nota) { case 0: console.log ( "Desaprobado" ); break; case 2: console.log ( "Aprobado" ); break; default: console.log ( "En espera" ); } FOR: for (contador=0 ; contador <=5 ; contador++) { console.log ( "Valor del contador = " + contador); } WHILE: while (i < 10) { console.log ( "i = " + i); i++; } DO WHILE: do { console.log ( "i = " + i); i++; } while (i < 10);

Objeto dom en JS

El modelo de objeto de documento (DOM) es una interfaz de programación para los documentos HTML y XML. Facilita una representación estructurada del documento y define de qué manera los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido.

¿Cómo trabaja?

Para crear el DOM, el navegador web que lee el archivo HTML toma todas sus partes, desde el elemento html raíz hasta las etiquetas span más pequeñas, y las devuelve como un objeto que el JavaScript comprende.
